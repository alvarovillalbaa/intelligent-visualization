# Convex Self-Hosting

Convex documents optional self-hosted operation. Hosted Convex remains the recommended path for most deployments.

Important notes:

- The Convex backend has its own licensing model; this repository must not bundle or relicense the Convex backend.
- Self-hosted Convex is different from this project’s native zero-hosted local SQLite mode.
- The default local mode remains `PERSISTENCE_PROVIDER=local`.

This guide will be expanded when `packages/provider-convex` is executable.
