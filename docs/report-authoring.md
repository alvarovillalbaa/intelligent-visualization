# Report Authoring

Reports combine narrative and dashboard-style blocks.

Current implementation:

- Report document schema.
- Starter report template.
- Artifact rendering for report blocks and accessible inline SVG charts.

Remaining work:

- DOM grid editor.
- Drag, resize, duplicate, lock, undo, and responsive breakpoint editing.
- Rich editor charts through shadcn/Recharts.
- Data import flows and immutable snapshots.
